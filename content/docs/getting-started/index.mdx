# Getting Started with CurryDash

CurryDash is a subscription-marketplace hybrid food delivery platform for the Sri Lankan diaspora
community in Victoria, Australia. The platform has three main repositories:

| Repository | Technology | Purpose |
|---|---|---|
| **Central Hub** | Next.js 16, TypeScript, Supabase | Dev monitoring dashboard, AI assistant, integrations |
| **Admin-Seller Portal** | Laravel 10.x, Filament 3.x, MySQL | Admin dashboard, vendor portal, backend APIs |
| **Customer App** | Flutter 3.x, Dart | Cross-platform iOS/Android/Web customer app |

## Prerequisites

- Node.js ≥20.9 (Central Hub)
- PHP 8.2+, Composer (Admin-Seller Portal)
- Flutter 3.x, Dart SDK (Customer App)
- Supabase account with project ID `hlcoyehcatbrqecjoclh`
- Jira Cloud account (CurryDash project)
- GitHub account with repo access
