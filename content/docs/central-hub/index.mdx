# CurryDash Central Hub

The Central Hub is a role-based project management and monitoring dashboard built with
Next.js 16. It serves four roles: Admin, Developer, QA, and Stakeholder.

## Key Features

- **Role-Based Access Control** — Three-layer RBAC (Edge Proxy → Server Components → Supabase RLS)
- **Dashboard Widgets** — Real-time data from Jira and GitHub with staleness indicators
- **Jira Integration** — Webhook pipeline, sprint tracking, issue management
- **GitHub Integration** — PR status, CI/CD pipeline, webhook processing
- **AI Assistant** — CopilotKit + Mastra backend with Anthropic Claude
- **Admin Panel** — System health, user management, dead letter queue

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 (App Router), React 18 |
| Language | TypeScript (strict mode) |
| Database | Supabase (PostgreSQL + Realtime + Auth) |
| Auth | Auth.js v5 (JWT sessions) |
| Styling | Tailwind CSS v4, shadcn/ui |
| State | Zustand 5 (UI state only) |
| AI | CopilotKit v1.51.3 + Mastra v0.10.15 |
| Integrations | Jira (jira.js), GitHub (Octokit) |
| Testing | Vitest (unit/integration), Playwright (E2E) |
